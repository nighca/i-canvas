Interactive-canvas
==================

## About

Lib for generate \& operate objects on canvas.

## Features

* DOM-like API

* Powerful plugins

* Easy to extend

* Lightweight

## Usage

	var canvas = new Canvas($('#canvas'), '2d'),
		doc = canvas.document,
		body = doc.body;

	var rec = doc.createElement('rectangle', {
		top: 100,
		left: 100,
		width: 100,
		height: 100,
		background: 'yellow',
		border: 'blue',
		'border-width': 5
	});

    body.appendChild(rec);
